# Scoring rules

## Model comparison criteria

| Code | Criterion | Weight |
|---|---|---:|
| C1 | Synchronisation multi-courbes | 0.25 |
| C2 | Clarté des ruptures | 0.20 |
| C3 | Robustesse des fenêtres | 0.20 |
| C4 | Parcimonie | 0.10 |
| C5 | Valeur ajoutée vs chronologie | 0.15 |
| C6 | Transférabilité vers 2011-2016 | 0.10 |

Raw criterion scores:

- 0 = non confirmé
- 1 = faible / narratif
- 2 = plausible
- 3 = robuste

## Automatic rejection

A model is rejected if:

1. its waves are visible in only one curve;
2. it depends on only one reference window;
3. it requires too many narrative exceptions;
4. it fails to distinguish 2007-2009 from 2010-2012 when claiming two cycles;
5. it adds no explanatory value beyond chronology.
