# Kuznets (15–25 years)

The infrastructure / demographic swing. Identified by Simon Kuznets (1930)
on US construction and immigration data; later reinterpreted through the
Lewis dual-economy framework as the structural-transformation cycle.

## What CPV measures

The CPV pipeline runs the four votant methods (D, E, F, G) on the Kuznets
band [15, 25] years. This is the band of long building cycles, demographic
transitions, and — in the modern reading — the **financial cycle**
(Borio & Drehmann 2009).

Model H (Borio financial cycle) is conditionally added to the votant pool
when sufficient credit + property data is available.

## Driver indicators

- `NE.TRD.GNFS.ZS` (trade % GDP) — long integration cycle.
- `SP.URB.TOTL.IN.ZS` (urban population, % total) — Lewis-Kuznets structural
  transformation proxy.
