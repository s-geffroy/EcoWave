# Anti-pseudoscience rules

Forbidden:

- forcing every crisis into five waves;
- accepting waves visible in only one variable;
- hiding missing sources;
- choosing reference windows after seeing the result;
- presenting exploratory outputs as final.

Required:

- competing counts;
- explicit rejection reasons;
- confidence grades;
- source manifest;
- raw values preserved;
- **a null/surrogate test on the champion's auto-computed evidence** (C1+C3 must
  beat random segmentations and time-scrambled surrogates — see
  `improvement_roadmap.md` #1);
- **a non-Elliott benchmark** (Model D, automatic change-point detection): if it
  matches the Elliott champion, Elliott claims no added structure (#2);
- **at least one pre-registered out-of-sample crisis** (holdout pilot) with
  phases/annotations frozen before normalisation (#3).
